执行代码审查时，必须严格遵循以下步骤：

1.  **实时读取文件**：不要依赖任何缓存或者上下文。
2.  **全量扫描与核查**：重点扫描：未清理的变量/字段/Import、冗余重复逻辑、过时/误导/无意义注释。
3.  **可视化输出规约（必须执行）**：
    - **摘要表格**：首先输出一个表格，包含：位置（行号）、类型、问题描述、建议操作、严重度（🔴高/🟠中/🔵低）。
    - **代码对比**：针对关键修改建议，提供 `❌ 修改前` vs `✅ 修改后` 的代码块对比。
    - **分级报告**：按严重程度排序，保持语言精炼，拒绝长篇大论。
